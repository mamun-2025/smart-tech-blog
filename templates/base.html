🧐 আপনার কোডে কী কী পরিবর্তন করলাম? (ব্যাখ্যা)
    ট্যাগ যোগ করা: বুটস্ট্র্যাপে nav-item বা nav-link গুলোকে সবসময় একটি
      (Unordered List) ট্যাগের ভেতরে রাখতে হয়। আপনি সরাসরি li লিখেছিলেন, যা HTML ভ্যালিডেশনের বাইরে। navbar-collapse: এটি যোগ করার ফলে যখন কেউ মোবাইল দিয়ে আপনার সাইট দেখবে, তখন মেনুগুলো একটি সুন্দর "হ্যামবার্গার মেনু" বাটনের ভেতরে ঢুকে যাবে। ms-auto: এটি ব্যবহারের ফলে আপনার ব্লগের নাম (Brand) থাকবে বামে এবং লিঙ্কগুলো চলে যাবে ডানপাশে, যা দেখতে আধুনিক লাগে। 💼 ইন্টারভিউ টিপস (Navbar নিয়ে) প্রশ্ন: নেভিগেশন বারে {% url %} ট্যাগ ব্যবহারের সুবিধা কী? উত্তর: এটিকে বলা হয় URL Reversing। আমরা যদি ম্যানুয়ালি /about/ লিখি এবং ভবিষ্যতে যদি কোনো কারণে ইউআরএল পরিবর্তন করি, তবে সব পেজে গিয়ে লিঙ্ক বদলাতে হবে। কিন্তু {% url 'blog:about' %} ব্যবহার করলে জ্যাঙ্গো অটোমেটিক urls.py থেকে সঠিক লিঙ্কটি খুঁজে নেয়। {% endcomment %}
